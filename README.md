# NR-12 SaaS

Plataforma multi-tenant para gestão de inspeções de máquinas e equipamentos
conforme a NR-12, com geração de laudos assistida por IA e coleta de dados
de campo via WhatsApp.

Este projeto é, ao mesmo tempo, um produto real e o veículo de aprendizado
de full-stack (React, TypeScript, Tailwind, shadcn/ui, Supabase, Stripe)
do autor. Está sendo construído em marcos, cada um desenhado para também
funcionar como uma gig isolada no Fiverr (database design, backend API,
automação n8n, frontend, pagamentos).

## Documentação

Toda decisão de arquitetura e todo desenho de dados fica documentado em
[`docs/`](docs/README.md) **antes** de virar código — não depois.
